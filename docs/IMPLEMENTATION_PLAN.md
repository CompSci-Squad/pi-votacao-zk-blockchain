> **For the agent:** Read this document from top to bottom before writing any code.

## Context and Objective

This repository is the **on-chain source of truth** for the entire voting system. It contains:

1. `VotingContract.sol` — the main election management contract.
2. `Verifier.sol` — auto-generated by SnarkJS from the PLONK circuit (imported from `pi-votacao-zk-circuits`).
3. A Python test suite (`pytest` + `web3.py`) that covers every contract function and security-critical path.
4. Hardhat scripts for local node management and contract deployment.

No relational database. No backend state. Everything critical lives on-chain.

---

## Fixed Architectural Decisions 🔒

| Decision | Value |
|---|---|
| Testing framework | **pytest + web3.py** (Ape Framework is explicitly rejected) |
| Local blockchain node | **Hardhat** (`npx hardhat node`, port 8545) |
| Solidity version | **0.8.x** (pin to a specific patch version in `hardhat.config.js`) |
| Verifier contract | Generated by SnarkJS from the circuits repo — **never edit by hand** |
| Election states | `PENDING → OPEN → FINISHED` — one-way transitions only |
| Nullifier storage | `mapping(uint256 raceId => mapping(uint256 nullifier => bool))` |
| Voter hashes | `uint256[] public voterHashes` — publicly readable for Merkle tree audit |
| Public signals order | `[merkleRoot, nullifierHash, candidateId, electionId, raceId]` — fixed by the circuit |
| `castVote` security pattern | **Checks-Effects-Interactions** — nullifier written before event emission |

## Phase 1 — Interface Definition

### 1.1 File: `contracts/IVerifier.sol`

Inspect the actual `Verifier.sol` generated by SnarkJS to determine the exact function signature. The PLONK verifier exported by SnarkJS typically uses this interface:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

interface IVerifier {
    /**
     * @notice Verifies a PLONK zero-knowledge proof.
     * @param pubSignals Public signals: [merkleRoot, nullifierHash, candidateId, electionId, raceId]
     * @param proof      Encoded PLONK proof bytes
     * @return           True if the proof is valid, false otherwise
     */
    function verifyProof(
        uint256[5] calldata pubSignals,
        bytes calldata proof
    ) external view returns (bool);
}
```

> **Agent note:** Check the actual signature in `Verifier.sol` before writing this interface. SnarkJS PLONK verifiers may use `uint256[]` vs `uint256[5]`. The interface must match exactly.

### 2.6 Admin Setup Functions (PENDING only)

```solidity
function addRace(string calldata name)
    external onlyAdmin inState(ElectionState.PENDING)
{
    races.push();
    races[races.length - 1].name = name;
}

function addCandidate(
    uint256 raceId,
    string calldata name,
    string calldata party,
    uint256 number,
    bytes32 photoHash
) external onlyAdmin inState(ElectionState.PENDING) {
    require(raceId < races.length, "Invalid race");
    Race storage race = races[raceId];
    uint256 candidateId = race.candidates.length;
    race.candidates.push(Candidate({
        id:        candidateId,
        name:      name,
        party:     party,
        number:    number,
        photoHash: photoHash,
        voteCount: 0
    }));
}

function registerVoterHashes(uint256[] calldata hashes)
    external onlyAdmin inState(ElectionState.PENDING)
{
    for (uint256 i = 0; i < hashes.length; i++) {
        voterHashes.push(hashes[i]);
    }
    emit VoterHashesRegistered(hashes);
}

function setMerkleRoot(uint256 root)
    external onlyAdmin inState(ElectionState.PENDING)
{
    voterMerkleRoot = root;
}
```

### 2.7 Election Lifecycle

```solidity
function openElection() external onlyAdmin inState(ElectionState.PENDING) {
    require(voterMerkleRoot != 0,   "Merkle root not set");
    require(voterHashes.length > 0, "No voters registered");
    require(races.length > 0,       "No races defined");

    state    = ElectionState.OPEN;
    openedAt = block.timestamp;

    bytes32 stateHash = keccak256(abi.encode(voterMerkleRoot, block.number));
    emit ElectionOpened(block.timestamp, stateHash);
}

function closeElection() external onlyAdmin inState(ElectionState.OPEN) {
    state    = ElectionState.FINISHED;
    closedAt = block.timestamp;
    emit ElectionClosed(block.timestamp, totalVotesCast);
}
```

### 2.8 `castVote` — Security Core (CEI Mandatory)

```solidity
function castVote(
    uint256 raceId,
    uint256[5] calldata pubSignals,
    bytes calldata proof
) external inState(ElectionState.OPEN) {

    // ── CHECKS ────────────────────────────────────────────────────────
    require(raceId < races.length,            "Invalid race");
    require(pubSignals[0] == voterMerkleRoot, "Merkle root mismatch");
    require(pubSignals[3] == electionId,      "Election ID mismatch");
    require(pubSignals[4] == raceId,          "Race ID mismatch");

    uint256 nullifier   = pubSignals[1];
    uint256 candidateId = pubSignals[2];

    require(!nullifiers[raceId][nullifier], "Nullifier already used");
    require(verifier.verifyProof(pubSignals, proof), "Invalid ZK proof");

    // ── EFFECTS ───────────────────────────────────────────────────────
    // 🔒 Nullifier MUST be written before any event emission
    nullifiers[raceId][nullifier] = true;
    totalVotesCast++;

    Race storage race = races[raceId];
    if (candidateId == 0) {
        race.blankVotes++;
    } else if (candidateId == 999) {
        race.nullVotes++;
    } else {
        bool found = false;
        for (uint256 i = 0; i < race.candidates.length; i++) {
            if (race.candidates[i].id == candidateId) {
                race.candidates[i].voteCount++;
                found = true;
                break;
            }
        }
        require(found, "Candidate not found");
    }

    // ── INTERACTIONS ──────────────────────────────────────────────────
    emit VoteCast(nullifier, raceId, candidateId);
}
```

### 2.9 Read Functions

```solidity
function getResults(uint256 raceId) external view returns (
    Candidate[] memory candidates,
    uint256 blankVotes,
    uint256 nullVotes
) {
    require(raceId < races.length, "Invalid race");
    Race storage race = races[raceId];
    return (race.candidates, race.blankVotes, race.nullVotes);
}

function getVoterHashes() external view returns (uint256[] memory) {
    return voterHashes;
}

function getRaceCount() external view returns (uint256) {
    return races.length;
}

function isNullifierUsed(uint256 raceId, uint256 nullifier)
    external view returns (bool)
{
    return nullifiers[raceId][nullifier];
}
```

## Known Issues and How to Handle Them

| Problem | Likely Cause | Action |
|---|---|---|
| `AssertionError: Hardhat node is not running` | Hardhat process not started | Run `npx hardhat node` in a separate terminal |
| `FileNotFoundError: Artifact not found` | Contracts not compiled | Run `npx hardhat compile` |
| `ContractLogicError` on valid proof | `IVerifier.sol` interface mismatch | Check `verifyProof` signature against actual `Verifier.sol` |
| `Merkle root mismatch` in tests | Test fixture using wrong root value | Use `open_election.functions.voterMerkleRoot().call()` to read on-chain value |
| `process_receipt` returns empty list | Wrong event name or ABI mismatch | Recompile and reload artifact; verify event name matches exactly |
| Skipped valid-proof tests | Proof fixtures not generated yet | Complete Phase 5 of `pi-votacao-zk-circuits` first |
